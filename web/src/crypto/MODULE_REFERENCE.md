# crypto/ — 客户端密码学核心

零知识架构的实现层。所有加密/解密在浏览器本地完成，服务器只是密文中转站。
一旦我所属的文件夹有所变化，请更新我。

## 文件清单

| 文件 | 地位 | 功能 |
|------|------|------|
| argon2.ts | 密码学链条第一环 | Argon2id 封装，从主密码+salt派生 32 字节 IKM |
| keys.ts | 密码学链条第二环 | HKDF-SHA256 密钥派生，IKM → MasterKey + AuthKey；附带 base64 工具 |
| vault.ts | 密码学链条第三环 | AES-256-GCM 加密/解密密码条目，每条独立 DEK |
