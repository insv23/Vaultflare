# Vaultflare — 功能待办清单

基于 2026-02-22 对后端 + Web 端的全面审查，以下是作为基础密码管理器仍需完成的功能。

---

## P0 — 没有就不像密码管理器

- [x] **密码生成器**
  - 用户创建/编辑密码时一键生成随机强密码
  - 可配置：长度、大小写、数字、特殊字符
  - 涉及：Web（CipherForm 内新增组件）

- [x] **修改主密码**
  - 用户注册后目前无法修改主密码
  - 需要：后端新增端点 + 客户端重新派生密钥并重新加密所有 DEK
  - 涉及：后端 + Web

- [x] **Toast 通知系统**
  - 目前只有 inline 错误，没有成功/操作反馈
  - 创建、编辑、删除、复制密码后用户看不到任何确认
  - 涉及：Web（引入 sonner 或类似库）

---

## P1 — 基础体验必需

- [ ] **密码强度指示器**
  - 注册页的主密码 + CipherForm 的密码字段
  - 展示强度等级（弱/中/强）
  - 涉及：Web

- [ ]  **CF 能否通过一个环境变量设置值来动态调节它是否可以继续注册**

- [ ] **导出 / 导入**
  - 导入：支持 CSV（兼容 Bitwarden/1Password 导出格式）
  - 导出：加密备份或明文 CSV（需二次确认）
  - 零知识架构下只能在客户端完成
  - 涉及：Web

- [ ] **深色模式切换**
  - CSS 变量（`.dark` 类）已定义，缺 UI 开关和 localStorage 持久化
  - 涉及：Web

- [ ] **版本冲突处理 UI**
  - 后端 409 conflict 已实现，前端只显示通用错误
  - 需要：提示"数据已被修改"并提供覆盖/刷新选项
  - 涉及：Web

- [ ] **删除账户**
  - 后端无端点（CASCADE 已就绪，只差接口）
  - 前端需要确认流程（输入主密码二次确认）
  - 涉及：后端 + Web

---

## P2 — 做了会好很多

- [ ] **排序功能**
  - 按名称、创建时间、最近修改排序
  - 涉及：Web

- [ ] **收藏 / 置顶**
  - 常用密码标记收藏，置顶显示
  - 涉及：后端（ciphers 表加字段）+ Web

- [ ] **分类 / 文件夹**
  - 密码分组管理，数量多时必要
  - 涉及：后端（新表或字段）+ Web

- [ ] **Session 过期友好提示**
  - Token 30 天过期后，前端收到 401 只跳登录页
  - 需要：Toast 提示"会话已过期，请重新登录"
  - 涉及：Web（依赖 Toast 系统）

- [ ] **速率限制**
  - 登录、注册端点无 rate limiting，暴力破解无阻碍
  - 涉及：后端（Cloudflare Workers 层面）
